# Traducción al Español (ES)

Archivos de traducción organizados por secciones para facilitar mantenimiento y legibilidad.

## Estructura de Archivos

- `common.json` - Textos comunes y de navegación
- `hero.json` - Sección principal (hero section)
- `music.json` - Selector de música
- `certifications.json` - Certificaciones y formación
- `info.json` - Títulos de secciones de información
- `technologies.json` - Tecnologías y skills
- `projects.json` - Proyectos (hacking y otros)
- `about.json` - Sobre mí (biografía e historia)
- `hobbies.json` - Pasatiempos e intereses

## Notas

Cada archivo es independiente y agrupa las traducciones por contexto temático.
El archivo `i18n.js` en la raíz combina todos estos archivos automáticamente.
