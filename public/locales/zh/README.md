# 中文翻译 (ZH)

按部分组织的翻译文件，便于维护和可读性。

## 文件结构

- `common.json` - 通用文本和导航
- `hero.json` - 主要部分（hero section）
- `music.json` - 音乐选择器
- `certifications.json` - 认证和培训
- `info.json` - 信息部分标题
- `technologies.json` - 技术和技能
- `projects.json` - 项目（黑客和其他）
- `about.json` - 关于我（传记和故事）
- `hobbies.json` - 爱好和兴趣

## 注意事项

每个文件都是独立的，按主题上下文对翻译进行分组。
根目录中的 `i18n.js` 文件会自动合并所有这些文件。
