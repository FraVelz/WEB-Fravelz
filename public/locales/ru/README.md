# Русские переводы (RU)

Файлы переводов, организованные по разделам для удобства обслуживания и читаемости.

## Структура файлов

- `common.json` - Общие тексты и навигация
- `hero.json` - Главный раздел (hero section)
- `music.json` - Выбор музыки
- `certifications.json` - Сертификаты и обучение
- `info.json` - Заголовки информационных разделов
- `technologies.json` - Технологии и навыки
- `projects.json` - Проекты (хакинг и прочее)
- `about.json` - Обо мне (биография и история)
- `hobbies.json` - Увлечения и интересы

## Примечания

Каждый файл независим и группирует переводы по тематическому контексту.
Файл `i18n.js` в корне автоматически объединяет все эти файлы.
